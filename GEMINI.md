# Project Overview

This is a personal portfolio website for Wayne Correa, a Cloud Infrastructure Product Leader. It's a static website built with HTML, CSS, and JavaScript. The site showcases his career achievements, expertise, and experience.

# Building and Running

This is a static website, so there is no build process. To run the website, simply open the `index.html` file in a web browser.

# Development Conventions

The code is well-structured and follows common web development best practices.

*   **HTML:** The HTML is semantic and well-organized.
*   **CSS:** The CSS is written using variables for colors and spacing, making it easy to maintain and update. It also includes responsive design for different screen sizes.
*   **JavaScript:** The JavaScript is used to add animations and interactive features to the site. The code is well-commented and easy to understand.
