Cloud Infrastructure Insights from PeeringDB

This project provides a tool to visualize and analyze interconnection data from PeeringDB, the source of truth for internet infrastructure. While PeeringDB is an excellent directory, it is not optimized for strategic analysis. This tool addresses that gap by answering complex questions about how networks, data centers, and cloud on-ramps connect globally.

It transforms raw, record-based data into an actionable map of the internet's physical topology. Users can explore which data centers are critical interconnection hubs, compare the private interconnect footprints of different providers, and identify strategic opportunities for network expansion. The project was born from a real-world need to move beyond manual, time-consuming data analysis and now serves as a public resource for infrastructure planners and network strategists.

  • Date: 2025-2026
  • Tools/Code: Python, AI-assisted data normalization, HTML, CSS, JavaScript
  • Website: cloudinfrastructure.org

Geo-Boundary Evidence Pack Generator

The GeoGuard Boundary Evidence Pack Generator is an AI-driven compliance automation platform developed as a capstone project for the AI Product Management class. Designed for networking professionals, it addresses a critical friction point in global data sovereignty: the "Technical Wall" between complex network telemetry and non-technical legal auditors.

In an era of tightening regulations like GDPR and various national data laws, multinational organizations must prove that sensitive traffic remains within specific sovereign borders. Currently, this requires network engineers to spend dozens of hours manually parsing cryptic BGP and NetFlow logs, a process that is reactive, error-prone, and often stalls enterprise sales cycles.

This project automates translating millions of technical data points into plain-English, auditor-ready executive narratives. Key features include cryptographic evidence signing for "verifiable trust" and anomaly forecasting to proactively identify routing risks.

The product strategy prioritizes data sovereignty through a tiered rollout: starting with a cloud-native beta and evolving into a localized, air-gapped virtual appliance in the future. This ensures that sensitive telemetry never leaves the customer's perimeter, satisfying the strictest security requirements of financial and government institutions. By reducing audit preparation time, the GeoGuard Boundary Evidence Pack Generator transforms network compliance from a technical bottleneck into a strategic business accelerator.

  • Date: 2026
  • Tools/Code: LLM/SLM (AI-driven), Path-to-Text Synthesis, Cryptographic Signing, BGP/NetFlow Analytics
